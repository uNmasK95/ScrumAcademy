The project

O projeto Scrum Academy surgiu na sequência de uma proposta da unidade curricular Gestão de Processos de Software do Curso de Engenharia Informática, na Universidade do Minho.
Foi proposta a realização de uma aplicação web capaz de auxiliar alunos e professores nos seus projetos em que é usada a metodologia ágil de software, nomeadamente o Scrum.
Desta forma, o projeto Scrum Academy procura permitir aos docentes e alunos universitários uma plataforma capaz de suportar esta tecnologia de forma mais facilitada do que as presentes no mercado. A ideia é que seja uma aplicação aberta e de fácil utilização, inserida num ambiente académico.

Our Team

A nossa equipa de desenvolvimento é constituída por estudantes do 4º ano do Mestrado Integrado em Engenharia Informática. É uma equipa jovem e com enorme vontande em adquirir novos conhecimentos. Este projeto é o primeiro realizado com as técnologias usadas e serviu de aprendizagem para trabalhos futuros.
